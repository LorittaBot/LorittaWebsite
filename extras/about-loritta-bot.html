Sobre a Loritta (Bot)

Como eu posso adicionar a Loritta no meu servidor?

Você pode adicionar a Loritta clicando aqui!

Para você conseguir me adicionar em um servidor, você precisa ter permissão de Administrador ou permissão de Gerenciar Servidor no servidor que você deseja adicionar!

Não está encontrando o servidor? Veja se a conta que você está logado no Discord de navegador é a mesma que está o servidor que você deseja me adicionar!

Quais são os comandos da Loritta?

Você pode ver todos os comandos que ela tem clicando aqui!

Qual é o prefixo da Loritta?

O prefixo padrão da Loritta é +, mas você pode alterar ele no painel de administração da Loritta. Se você quer descobrir o prefixo dela em um servidor que você está, mencione ela no chat que ela irá responder o prefixo no servidor que você mencionou ela!

Como altero a linguagem da Loritta?

Você pode alterar a linguagem que a Loritta fala utilizando +linguagem

Como posso configurar/personalizar a Loritta?

Você pode configurar ela no painel de administração!

Tenho uma dúvida sobre a Loritta! Aonde posso perguntar?

No nosso servidor de suporte!

Como altero o avatar da Loritta?

É impossível alterar o avatar dela, apenas quem criou ela pode alterar, e, ao alterar, o avatar é alterado para todos que usam a Loritta... Não iremos alterar o avatar dela só porque você quer colocar uma imagemzinha tosca com o logo do seu servidor.

Quais são as variáveis/placeholders que eu posso usar em mensagens do painel?

Existem vários placeholders que você pode usar nas mensagens da Loritta! Veja a lista clicando aqui!

Porque a Loritta existe?

Ela existe por vários motivos, mas aqui vai alguns deles...

  • Eu queria refazer o meu antigo bot (SparklyBot) que eu tinha feito apenas para diversão (ele era muito bagunçado, já que foi o meu primeiro bot), ele era um bot com vários comandos de entreterimento (muitos deles foram adicionados na Loritta! Como o treta news, sistema de anti-flood, vaporwave, sistema de música e muito mais!) e com algumas funções básicas como mensagens de entrada/saída.
  • A @Pandinhaaa queria um bot para o servidor do Discord dela.
  • Na época eu não conhecia nenhum bot brasileiro decente para o Discord e, na época, várias pessoas brasileiras estavam começando a migrar para o Discord.

Então eu resolvi criar a @Loritta!

Quando a Loritta foi criada?

Ela começou a ser desenvolvida em 30 de Março de 2017, mas ela só ficou pública para qualquer um adicionar em 12 de Abril de 2017.

Eu posso fazer vídeos, tutoriais, desenhos, histórias com a personagem e o bot Loritta?

Sim! Você tem total direito de fazer vídeos, tutoriais, desenhos/fan arts, histórias e muito mais com a Loritta! Se você quiser, você pode mostrar o seu trabalho para a nossa equipe no servidor de suporte.

A Loritta está falando sozinha no meu servidor!

Antes de você ficar "jesus me salva a lori foi hackeada tira ela do servidor ", existem vários motivos dela poder estar falando assim.

Se você ver uma mensagem "estranha" vinda dela, pode ser várias coisas, como por exemplo:

  • Alguém usando o comando +say, é possível você enviar mensagens em um canal usando o +say até em outros canais de texto (+say #canal-de-texto Eu sou muito fofa!), sendo assim possível "trollar" outras pessoas deste jeito.
  • Alguém usando a função de "Testar Mensagem" no painel dela. Para facilitar a criação e customização de mensagens, existe um botão no painel dela para testar a sua mensagem criada, para você ver como fica no Discord. Infelizmente algumas pessoas gostam de "abusar" desta função usando ela para mandar mensagens como se fosse a Loritta.
  • Alguém usando uma webhook que tem o mesmo avatar e nome. Para descobrir se é uma webhook, clique no avatar da webhook, se não aparecer nenhuma informação (apenas o avatar e o nome), quer dizer que é uma webhook.
  • Alguém criou um bot com o mesmo avatar e nome. Para descobrir se é um bot falso, veja se o nome e o discriminator é Loritta#0219, se não for, quer dizer que não é a Loritta!

Qual linguagem a Loritta usa, e qual library?

Português, inglês, espanhol... ah, calma, não é isto que você quer saber, né?

Originalmente ela foi criada em Java (utilizei o Eclipse para programar), mas alguns meses depois de eu ter criado ela, eu fiquei interessado sobre Kotlin, então resolvi começar a migrar ela para Kotlin!

Atualmente ela é feita usando Java e Kotlin (utilizando o IntelliJ IDEA para programar), usando a incrível library JDA para interagir com a plataforma Discord.
