Sobre o Discord

Discord? É de comer?

Discord é uma plataforma de chat, voltado para comunidades de jogos. Atualmente o Discord está disponível em várias plataformas, como Windows, Mac OSX, Android, iOS, Linux e até mesmo em navegadores web, sendo possível usar o Discord sem precisar baixar nada!

Hoje em dia, o Discord é utilizado como uma alternativa ao TeamSpeak 3 e ao Skype e suporta mais de 25 milhões de usuários!

Nomenclaturas comuns de se usar no Discord

Nomenclatura Significado
Guild Servidor
Embed Embeds são mensagens que ficam dentro de uma "caixinha" no Discord, sendo possível ter mais customização sobre elas, como alterar a cor lateral, colocar imagens, thumbnails e muito mais! Apenas bots conseguem enviar tais mensagens. (mas é possível enviar como se fosse um usuário, mas isto é contra os termos de uso do Discord e você poderá ser suspenso!)
Discord Nitro Ao comprar Nitro, você ajuda o Discord a ficar online e ganha algumas vantagens exclusivas apenas para Nitro, por exemplo: Usar emojis de qualquer servidor em qualquer servidor, usar emojis animados, melhor qualidade ao compartilhar tela e muito mais!
Discord Hypesquad Hypesquad é uma equipe de pessoas que tem o objetivo de divulgar o Discord para novas pessoas.
Discord Partner Discord possuem parceiros, pessoas famosas fora da plataforma (como YouTubers, streamers, etc) que trabalham para trazer novas pessoas para a plataforma.
ID ID é um valor único para algo. No Discord, cada entidade (mensagens, usuários, servidores, etc) possuem um ID, você pode pegar os IDs de tais coisas ativando o "Modo de Desenvolvedor" nas configurações de aparências no Discord.
DM/MD/PV "Mensagens diretas", mensagens enviadas diretamente para uma pessoa. (Botão direito -> Mensagem)
Bot (Ro)bot, robô. No Discord, "robôs" se chamam "bots".
Bot List Lista de bots. Existem várias bot lists online para que você possa ver e encontrar novos bots para o seu servidor!
Bot Collection Um servidor que é utilizado apenas para adicionar o máximo número de bots possível. Muitos bots automaticamente saem de tais servidores já que é utilizado muitos recursos para processar um servidor que possui quase nenhum membro.
Shard Separar várias instâncias de um determinado aplicativo, para diminuir o uso de recursos e poder separar o aplicativo em várias máquinas para processar mais requests simultâneos. No Discord, todos os bots que estão em mais de 2500 precisam usar shards, cada shard pode controlar, no máximo, 2500 servidores. (se você tem um bot e ele não faz sharding, ele não poderá ser adicionado em mais de 2500 servidores). Uma shard seria como se você abrisse o Discord novamente, só que cada novo "discord aberto" possuísem servidores diferentes.
ToS Termos de Uso. No Discord, normalmente é usado para significar os termos de uso do Discord.
