Placeholders & Variáveis

Existem vários placeholders que você pode usar nas mensagens da Loritta!

Geral

Placeholder Significado
{user} Mostra o nome do usuário que provocou a ação
{@user} Menciona o usuário que provocou a ação
{nickname} Mostra o nome do usuário no servidor (caso ele tenha mudado o apelido dele no servidor, irá aparecer o apelido dele, em vez do nome original)
{user-discriminator} Mostra o discriminator do usuário que provocou a ação
{user-id} Mostra o ID do usuário que provocou a ação
{user-avatar-url} Mostra a URL do avatar do usuário
{guild} Mostra o nome do servidor
{guildsize} Mostra a quantidade de membros no servidor

Moderação

Placeholder Significado
{reason} Motivo da punição, caso nenhum motivo tenha sido especificado, isto estará vazio
{punishment} Punição aplicada (ban, mute, kick, etc)
{staff} Mostra o nome do usuário que fez a punição
{@staff} Menciona o usuário que fez a punição
{staff-discriminator} Mostra o discriminator do usuário que fez a punição
{staff-id} Mostra o ID do usuário que fez a punição
{staff-avatar-url} Mostra a URL do avatar do usuário que fez a punição

Loritta's Server List

Placeholder Significado
{vote-count} Mostra quantos votos o usuário que votou possui
